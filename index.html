System Instructions

1. Dashboard Navigation

Assessment Architect: Enter this module to author MCQ questions, format text, define scoring, and securely export your professional assessments.

Descriptive Architect: Enter this module to author essay questions and set up grading rubrics. You can mix both types in the same vault.

Strategic Ledger: A dedicated daily quick-journal. Your entries auto-save immediately to your browser.

2. Authoring & Formatting

Use the Text Style buttons to apply bold, italic, or underline formats. Utilize the List controls to automatically generate perfectly aligned nested lists within any question, explanation, or case study.

Text Alignment: Highlight text and use the Alignment buttons (L, C, R, J). This will inject tags like [align-center] which the engine will perfectly render as aligned blocks on the output screen.

3. Advanced Input Tools

OCR Scanner: Click the scan button to upload an image and seamlessly extract its text directly into the active text area.

Custom Tables: Deploy Standard Tables for regular grids or Pro Tables for advanced accounting formats.

4. Security & Deployment

Data Persistence: The application autosaves locally using IndexedDB. To securely back up your data or distribute an exam, click EXPORT SECURE and define a password. Use Import File to load pre-built exam structures.
